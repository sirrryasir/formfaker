# FormFaker Roadmap

## Vision
To be the **Docker of Dummy Data**: The standard, offline-first tool developers trust to test forms with realistic data.

---

## Phase 1: Foundation (Completed)
- [x] **MVP Core**: Smart scanning, context detection, random filling.
- [x] **Architecture**: Modular content scripts, profile consistency.
- [x] **Basic Locales**: USA (US) and Somalia (SO).
- [x] **Settings**: Toggles for safety and localization.

## Phase 2: Global Expansion (Completed)
- [x] **50+ Locales**: From Argentina to Vietnam, we now cover the globe.
- [x] **Combinatorial Generation**: Millions of unique profiles locally.
- [x] **Zero Dependencies**: 100% Offline.

## Phase 3: Growth & Community (Current)
**Goal**: Build trust and adoption through open source.

- [ ] **Store Launch**: Chrome Web Store submission.
- [ ] **Community Contributions**: Streamline PR process for new locales.
- [ ] **Ambassador Program**: Local maintainers for specific country data.

## Phase 4: Advanced Features (Future)
- [ ] **Seeded Randomness**: Reproducible bugs (`seed=123`).
- [ ] **Team Configs**: Share exclude rules via config file.
- [ ] **API Support**: `window.formfaker.fill()` for E2E testing (Cypress/Playwright).
